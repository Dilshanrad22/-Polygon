# Asset Placeholder

This folder contains app assets:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen (1284x2778)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

## Creating Assets

You can use the SVG files as templates or create your own PNG images.

For quick setup, you can use placeholder images or online tools like:
- https://www.figma.com
- https://www.canva.com
- https://placeholderlogo.com

| Asset | Size | Format |
